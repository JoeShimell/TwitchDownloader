
# YouTube Chat Renderer
**Credit to [lay295](https://github.com/lay295) the original author**
Modified version of [TwitchDownloader](https://github.com/lay295/TwitchDownloader) that supports rendering youtube chat jsons created with [chat-downloader](https://github.com/xenova/chat-downloader).
Changes were made quickly to get it up and running, this won't be perfect.
